QUICK START GUIDE
==================

IMPORTANT: Which file to run?
------------------------------
✅ ALWAYS double-click: switch_resolution.bat
❌ DO NOT double-click: .ps1 files (they will open as text files)

Step 1: Setup (First time only)
-------------------------------
Double-click "setup.bat" - it will download NirCmd automatically

Step 2: Use it
--------------
Double-click "switch_resolution.bat" to switch resolutions!

That's it! The window will close automatically.

Need help?
----------
See README.md for detailed instructions.
